Description

A collection of 29 vector charts for infographics, statistics reports, dashboards, quickstats and other data visualisation needs.

  • All shapes are vector (fully scalable)
  • All chart bars (displaying values from 0% to 100%) are pre-made, easy to change them
  • Easy to edit colors, size and thickness of the bars
  • Layered, grouped and well-organized PSD
  • Included:
    • 29 PSD files
    • Very basic documentation (readme.txt)
    • JPG files (for preview)

Notes
