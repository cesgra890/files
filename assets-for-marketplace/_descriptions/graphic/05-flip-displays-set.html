A set of 12 flip displays suitable for displaying days, time, downloads, visitors, prices, event dates (for example, website launch) or any other information. Displays: short date, full date, two counters and two clocks (all in two colors and on two different backgrounds). In addition all these displays are made in 3 sizes: large, medium and small. There’s also a design for highlighting day in the date displays.

  • All elements are on separate layers which are grouped into layer groups;
  • The font used in the displays is SV Basic Manual. It’s free, so you won’t need to spend extra money on buying the font. You can download it from www.dafont.com
  • Easy editable text;
  • You can use any other font. Works best with monospaced (fixed-width) fonts;
