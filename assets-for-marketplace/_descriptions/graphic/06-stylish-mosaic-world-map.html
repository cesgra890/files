Stylised world map made from five simple repetitive shapes. The file contains six color variations of the map.

Technical info:

  • All elements are 100% vector outlines, so it’s possible to scale the globes to any size without losing quality.
  • Elements are grouped into labeled layers, it’s easy to edit the illustrations;
  • Global colors used, so it’s easy to edit colors in whole illustration;
  • No gradient mesh used;
  • The original map is taken from the NASA website and then traced;
